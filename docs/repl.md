# REPL

Use this page when you want to poke the runtime interactively without building a whole application first.

## Main Entry Point

The facade re-exports `run_demo_loop`, which is the quickest way to wire an agent into an interactive loop.

## When The REPL Is Useful

- testing instructions
- trying tool visibility changes
- validating handoffs
- smoke-testing session behavior

## What It Is Not

The REPL is a development tool. It is not the production orchestration surface for your application.
