# Realtime Events

Use this page when you want to understand what flows through a realtime session as it runs.

## Event Families

- transcript deltas
- raw model events
- tool lifecycle events
- handoff events
- interruption events
- session updates and close events

## Design Rule

Treat realtime events as the live operational contract of the session. Do not rebuild your own second event taxonomy on top of them unless you have a product-specific reason.
